# ATCommandInterface
Allows an Arduino to respond to at commands as if it were a modem.


This is a very basic version with only a hand full of demo commands as examples.


Examples:

AT Command Receiver

Version 2016/05/06 0.000

OK

&nbsp;

AT+HELO

Helo AT Reader

OK

&nbsp;


AT+RAND

+RAND=7

AT+RAND=999

+RAND=438

OK

&nbsp;


AT+NULL

OK

&nbsp;


AT+STAT

+STATUS=A OK!

AT+STA

A OK!
